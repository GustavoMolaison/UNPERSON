**UnPerson** is a story-driven deduction and pattern recognition game, set in a world enduring an unimaginable catastrophe and a psychological nightmare.

The player embodies a specially selected detective-judge whose job is to resolve various cases, identify culprits, and expose "UnPersons"—individuals whose memories have been altered and who behave in unpredictable ways. 

If that wasn't enough, the protagonist must navigate an authoritarian, heavily surveilled society and workplace. They will be forced to make tough decisions, constantly weighing the lives of those close to them against their own morality.

The project remains in a very early development phase, and not all core mechanics are fully implemented yet.
